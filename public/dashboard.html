System Flow

  1. ESP32 detects a fall and inserts a row into fall_events.
  2. The patient phone page receives the new event with Supabase realtime.
  3. The phone fetches exact GPS coordinates using navigator.geolocation.
  4. The phone inserts latitude and longitude into locations.
  5. This dashboard refreshes automatically and shows the latest health and map data.
ESP32 + MPU6050 | v Supabase fall_events (heart_rate, spo2, time) | v Phone browser realtime listener | v Exact GPS fetch with navigator.geolocation | v Supabase locations (latitude, longitude) | v Live dashboard + map auto-update
